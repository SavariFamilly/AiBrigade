Bot Skins Directory
===================

Place bot skin textures (PNG files) in this directory.

Skin Format:
- 64x64 pixels (same as player skins)
- PNG format with transparency
- Follow Minecraft player skin format

Example skins to create:
- bot_soldier_1.png
- bot_scout_1.png
- bot_guard_1.png
- bot_medic_1.png
- bot_engineer_1.png

The mod will randomly select from available skins when spawning bots.
You can also specify a specific skin via commands.

For more information on creating Minecraft skins, visit:
https://minecraft.fandom.com/wiki/Skin
